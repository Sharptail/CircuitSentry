# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-06-28

### Added
- Initial public release of CircuitSentry.
- Windows overlay UI displaying CPU/GPU temperature and usage.
- Tray icon support with restore and exit controls.
- Options for always-on-top, transparency, and color customization.
- Admin elevation handling for reliable sensor access.
- Elevated startup support via Windows Task Scheduler.
- Improved README with user-friendly executable installation instructions.
- Added changelog documentation.
